Cara penggunaan API endpoint:

  • Untuk mendapatkan data gedung UPI: GET /api/buildings
  • Untuk mendapatkan data gambar gedung UPI: GET /api/:buildingName/images
  • Untuk mendapatkan rute dan jarak tempuh antara dua gedung UPI: GET /api/buildings/direction?start=TEMPAT_AWAL&destination=TEMPAT_AKHIR

Contoh penggunaan:

  • /api/buildings
  • /api/REKTORAT BUMI SILIWANGI/images
  • /api/buildings/direction?start=REKTORAT BUMI SILIWANGI&destination=MUSEUM PENDIDIKAN

Semua response dalam bentuk JSON:

{ "_id": "CONTOH_ID", "name": "NAMA_GEDUNG", "description": "CONTOH_DESKRIPSI", "coordinate": [ "LONGITUDE", "LATITUDE" ] }
